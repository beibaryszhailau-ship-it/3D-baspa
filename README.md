# 3D Oqulyq — HTML-only AI Assistant

Бұл нұсқада AI ассистент сайттың өзінде жұмыс істейді. Сервер, npm start, API кілт қажет емес.

Іске қосу:
1. `index.html` файлын браузермен ашыңыз.
2. Тіркеліп немесе аккаунтқа кіріңіз.
3. Оң жақтағы 🤖 батырмасын басып, ассистентті қолданыңыз.

Ескерту: Бұл қауіпсіз HTML-only ассистент. Ол OpenAI API қолданбайды, дайын ережелер бойынша жауап береді.
