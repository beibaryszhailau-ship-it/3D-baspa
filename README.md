# 3D Баспа | 6-сынып Информатика электрондық оқулығы

Бұл жоба — 6-сынып оқушыларына арналған **«3D Баспа»** тақырыбындағы электрондық оқулық. Сайт HTML, CSS және JavaScript арқылы жасалған. Оқулықта 3D баспа, 3D модельдеу және SketchUp бағдарламасы туралы теориялық материалдар, тест тапсырмалары, практикалық жұмыстар және интерактивті бөлімдер қамтылған.

## Жобаның мақсаты

Оқушыларға 3D баспа технологиясы мен 3D модельдеу негіздерін түсінікті, көрнекі және интерактивті түрде үйрету. Электрондық оқулық арқылы оқушы теорияны оқып, тест тапсырып, практикалық тапсырмалар орындап, оқу прогресін бақылай алады.

## Сайт бөлімдері

- **Бастапқы бет** — 3D модель қарау, 3D модель енгізу, 3D баспа жаңалықтары және суреттер галереясы.
- **Кіріспе** — оқулықтың мақсаты мен мазмұны туралы ақпарат.
- **Теория** — 3D редактор, SketchUp құралдары және 3D объектілердің модельдерін құру тақырыптары.
- **Тест** — 15 тапсырмадан тұратын қорытынды тест.
- **Практика** — SketchUp бағдарламасы, бейнежазбалар, LearningApps және Wordwall тапсырмалары.
- **Автор** — электрондық оқулық авторы туралы ақпарат.
- **AI ассистент** — сайт ішінде жұмыс істейтін көмекші ассистент.

## Негізгі мүмкіндіктер

- Қараңғы стильдегі заманауи дизайн
- Бір бет ішінде бірнеше бөлімнің ашылуы
- 3D модельді қарау мүмкіндігі
- `.glb` форматындағы 3D модельді енгізу
- Тіркелу және аккаунтқа кіру функциясы
- Аккаунттан шығу батырмасы
- Оқу прогресін бақылау
- Теория және тест кезеңдері бойынша прогресс
- Интерактивті тест тапсырмалары
- Drag-and-drop арқылы реттілік тапсырмасы
- SketchUp бойынша бейнежазбалар
- LearningApps және Wordwall тапсырмалары
- HTML ішінде жұмыс істейтін AI ассистент
- GitHub Pages арқылы жариялауға ыңғайлы

## Оқу прогресі

Сайтта оқу прогресі 2 кезеңнен тұрады:

1. **Теория өту** — 50%
2. **Тест тапсыру** — 50%

Оқушы теория бөліміне кірген кезде прогресс 50% болады. Тесттен кемінде 10 балл жинаса, тест кезеңі орындалып, прогресс 100% болады.

## AI ассистент туралы

Сайтқа HTML ішінде жұмыс істейтін AI ассистент қосылған. Ол 3D баспа, SketchUp, 3D модельдеу, тест және практика бойынша дайын жауаптар береді. Бұл нұсқа API кілтсіз жұмыс істейді, сондықтан GitHub Pages-та қауіпсіз іске қосылады.
